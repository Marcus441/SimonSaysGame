# SimonSaysQUTy
A Simon Says game developed on a custom QUTy board fitted with an AVR chip, USB-C power, UART and programming (UPDI) interface, 4 pushbuttons, piezo buzzer element, potentiometer and a 2-digit 7-segment LED display. Built using the C programming language.
### Required Functionality:
-	Adjust the difficulty of the game by increasing or reducing playback delay using the potentiometer.
-	Interface with the games sequence delivered via the LED display and buzzer tone through UART or pushbuttons directly.
-	Score is displayed 
### Technologies:
-	Timers, delays and vector interrupts
-	UART write and input options.
-	Debouncing
-	State machines
